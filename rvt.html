Cálculo da Revisão da Vida Toda (RVT) - Introdução ​

A Revisão da Vida Toda (RVT) trata da possibilidade de recálculo da renda mensal inicial (RMI) dos benefícios previdenciários concedidos a partir da publicação da Lei 9.876/99 com base na aplicação da regra permanente (conforme artigo 29, incisos I e II, da Lei nº 8.213/91) quando favorável ao segurado (Tema 1102 do STF).

Acesso à ferramenta de cálculo ​

A ferramenta de cálculo da Revisão da Vida Toda (RVT) está disponível no link: https://www.trf3.jus.br/cecalc/rvt

Interface ​

A página inicial apresenta a seguinte interface:

Menu Cálculo ​

O menu Cálculo apresenta as seguintes funcionalidades:

  • extração dos dados da Carta de Concessão e do extrato CNIS (Relações Previdenciárias e Remunerações);
  • identificação automática dos dados acrescidos e/ou modificados pelo usuário (Salários de contribuição e Vínculos);
  • em conformidade com todos os critérios de validação fixados pelo GT do Conselho da Justiça Federal;
  • comparação de resultados nos casos de cálculo com direito adquirido.

Existem 3 métodos disponíveis para realizar a importação dos dados:

  • "arrastar" os arquivos do computador;
  • copiar o conteúdo dos arquivos e utilizar o botão Caixa de Texto;

tutorial

"Arrastando" arquivos do computador ​

Para realizar este procedimento, basta clicar com o mouse no arquivo gravado no computador e arrastá-lo para o quadro respectivo (Carta de Concessão ou extrato CNIS/SAT):

Copiando conteúdo dos arquivos com o botão Caixa de Texto ​

Passos para realizar o procedimento:

  • os dados devem ser selecionados e copiados de um arquivo válido;
  • clicar no botão Caixa de Texto;
  • clicar na área de transferência e colar os dados.

Copiando conteúdo dos arquivos com o botão Área de Transferência ​

Passos para realizar o procedimento:

  • os dados devem ser selecionados e copiados de um arquivo válido;
  • clicar no botão Área de Transferência.

Aba Salários ​

Na aba Salários é possível inserir/alterar os salários de contribuição, bem como adicionar a Fonte dos respectivos valores. Isso pode ser feito nas colunas Modificadorese Fonte:

❗ IMPORTANTE

A planilha só permite inserção de dados nas competências em que há vinculos com a Previdência Social. Os períodos podem ser visualizados/adicionados/alterados na aba Vinculos

Teclas de atalho ⚡ ​

Recomenda-se o uso do teclado para navegação e inserção de dados na aba Salários, tendo em vista a disponibilidade das seguintes teclas de atalho:

💡 DICA

É possível copiar dados da ferramenta para uma planilha Excel (e vice-versa). Caso os dados estejam sendo copiados da ferramenta, há opção para cópia dos valores com "cabeçalhos" (acessível clicando com botão direito do mouse e depois em Copiar com cabeçalhos)

Exportando um arquivo .csv ​

É possível exportar todos os dados da aba Salários para um arquivo .csv (comma separated values: valores separados por vírgulas). Para isso, basta clicar com o botão direito do mouse e selecionar a opção Baixar arquivo ".csv"

Aba Vínculos ​

Na aba Vínculos é possível inserir/alterar períodos de vínculos com o INSS.

Os dados já são preenchidos quando é realizada a importação do extrato CNIS/SAT, podendo ser alterados livremente:

Também é possível incluir novos períodos, clicando no botão + Adicionar Vínculos e preenchendo os dados:

Aba Demonstrativo ​

Na aba Demonstrativo, são demonstrados os resultados do cálculo elaborado, podendo ser impresso ou gerado um arquivo PDF. Os dados são separados nos seguintes blocos:

  1. Resultado
  2. Dados do Segurado e do Benefício
  3. Parâmetros Gerais e Valores Calculados
  4. Relação dos salários-de-contribuição
  5. Moedas e Encadeamento
  6. Modificações

Resultado ​

Aqui são demonstrados os valores das Rendas Mensais Iniciais (RMIs) apurada administrativamente e recalculada com base nos dados inseridos. Também é demonstrada a diferença entre os valores:

Dados do Segurado e do Benefício ​

Neste bloco, temos os dados referentes ao segurado e ao benefício:

Parâmetros Gerais e Valores Calculados ​

Os parâmetros considerados para a elaboração do cálculo são listados nesta parte. Além disso, temos também os valores calculados que resultaram na RMI revista.

Relação dos salários-de-contribuição ​

Neste bloco, são apresentados os valores de todos os salários de contribuição (original e atualizado) utilizados no cálculo da RMI são listados, bem como o índice de atualização para cada competência.

Moedas e Encadeamento ​

Aqui é demonstrada uma tabela com todas as alterações de moeda, bem como o encadeamento de índices utilizado no recálculo do valor do benefício.

Modificações ​

Por fim, neste bloco temos todos os dados que foram inseridos manualmente pelo usuário (ou seja, estão excluídos os dados vindos da Carta de Concessão e do CNIS).

Menu Carta de Concessão e Menu CNIS ​

Os menus Dados Carta de Concessão e Dados CNIS apresentam as informações e valores que foram identificados nos documentos importados, sendo possível detalhar todas as informações que foram trazidas para a ferramenta.

Botão Definir parâmetros de cálculo ​

Clicando nesse botão, é aberto um menu com algumas opções de configuração do cálculo:

  1. Encadeamento
  2. Aplicar IRSM
  3. Conflito nos salários
  4. Usar salário-mínimo na ausência de salários

Encadeamento ​

Tendo em vista os encadeamentos de índices mais utilizados, existem 3 opções de encadeamento. As diferenças ocorrem apenas na atualização dos salários de contribuição anteriores a outubro/88.

  1. INPC SIDRA: índice compatibilizado após a mudança de sistemática de apuração.
  2. OTN/ORTN.
  3. INPC Original: índices divulgados sem compatibilização após a sistemática de apuração.

Aplicar IRSM ​

Possibilidade de não aplicar o índice referente ao IRSM de fevereiro/94 (39,67%).

Conflito nos salários ​

Opção para alterar os valores que devem ser utilziados no caso de valores diferentes entre a Carta de Concessão e o CNIS.

Usar salário-mínimo na ausência de salários ​

Opção para deixar de utilizar o valor do salário-mínimo nos períodos em que não constam salários nos períodos lançados na aba Vínculos

Opções Salvar, Abrir e Novo ​

No canto superior direito da ferramenta, temos 3 botões:

  1. Salvar;
  2. Abrir;
  3. Novo.

Salvar ​

Clicando no botão Salvar, é baixado um arquivo .cecalc no computador, contendo todos os dados do cálculo efetuado, possibilitando sua recuperação posteriormente.

Abrir ​

Clicando no botão Abrir, será aberta uma caixa de texto, em que é possível selecionar um arquivo .cecalc salvo anteriormente. Dessa forma, o cálculo será recuperado e aberto na ferramenta.

Novo ​

Clicar no botão Novo, surgirá a seguinte mensagem:

Clicando em Confirmar, todos os dados inseridos serão apagados.

💡 DICA

Ao iniciar um cálculo novo, é altamente recomendável que seja utilizado o botão Novo, garantindo que os dados de outro cálculo não sejam utilizados incorretamente.

❗ IMPORTANTE

Quando o navegador for fechado por algum motivo (desligamento do computador, por exemplo) e a página da ferramenta for novamente acessada, surgirá uma mensagem referente à existência de dados carregados, sendo possível continuar um cálculo que estava sendo realizado anteriormente. Outra opção é escolher a opção Novo Cálculo, que possui comportamente semelhante ao botão Novo

Menu ​

No menu localizado no canto superior direito, temos as seguintes opções:

  1. Tutoriais e ajuda;
  2. Problemas e sugestões;
  3. GACO;
  4. CECALC;
  5. Fábrica de Cálculos.

Tutoriais e ajuda ​

Clicando nesta opção, o usuário será redirecionado para este site, com informações e tutoriais para utilização das ferramentas de cálculo.

Problemas e sugestões ​

Nesta opção, o usuário tem acesso a um formulário, que pode ser utilizado para reportar problemas, bem como dar sugestões referentes às ferramentas.
